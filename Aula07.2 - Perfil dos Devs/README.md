# Tarefa: Perfil dos Devs
Este app foi desenvolvido com ajuda da API pública do GitHub (https://api.github.com/users/)

Para a tarefa, foi requisitado:
  - Um Input para inserção do nome do login de algum usuário
  - Um botão para buscar o login na API do GitHub
  - Dados da API: name, id, avatar_url, followers, following e public_repos.

Também foram adicionados
  - Caso pressione no ícone de link externo, ele irá abrir o link do perfil do usuário, que também é pego nos dados da API.
